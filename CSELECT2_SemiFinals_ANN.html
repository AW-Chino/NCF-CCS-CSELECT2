Activation Functions

What are Activation Functions?

Activation functions are mathematical equations that determine the output of a neural network. They decide whether a neuron should be activated ("fired") or not based on whether each neuron's input is relevant for the model's prediction.

Why Do We Need Activation Functions? ü§î

  • Non-linearity: They help neural networks learn complex patterns by introducing non-linear properties
  • Decision Making: Help determine how important each neuron's input is
  • Output Range: Normalize outputs to a specific range (like 0 to 1)

ReLU (Rectified Linear Unit)

ReLU Function

Best for: Hidden layers

Mathematical Form:

f(x) = max(0, x)

When to Use:
  • Default choice for hidden layers
  • Deep neural networks
  • Convolutional Neural Networks (CNN)
Example:
Input Output Explanation
2.0 2.0 Positive passes through
-1.0 0.0 Negative becomes zero
3.5 3.5 Positive passes through
Advantages:
  • Computationally efficient
  • Helps with vanishing gradient
  • Creates sparse activations
Disadvantages:
  • "Dying ReLU" problem
  • Not zero-centered

Tanh (Hyperbolic Tangent)

Tanh Function

Best for: Normalized data

Mathematical Form:

f(x) = (e^x - e^-x) / (e^x + e^-x)

When to Use:
  • Hidden layers
  • When zero-centered output is needed
  • For data normalized between -1 and 1
Example:
Input Output Explanation
2.0 0.96 Strong positive
0.0 0.00 Neutral
-2.0 -0.96 Strong negative
Advantages:
  • Zero-centered
  • Stronger gradients than sigmoid
  • Bounded output
Disadvantages:
  • Still has vanishing gradient
  • Computationally expensive

The Learning Process üéì

How Neural Networks Learn

Neural networks learn through a process called training, where they gradually improve their accuracy by adjusting their internal parameters based on examples. Think of it like learning to cook: you try, taste, adjust, and try again until you get it right.

The Learning Cycle Explained

1. Predict
  • Network processes input through layers
  • Uses current weights and biases
  • Makes initial guess based on parameters
2. Compare
  • Compares prediction with actual answer
  • Calculates error using loss function
  • Determines how far off the prediction was
3. Adjust
  • Uses backpropagation to find which weights to change
  • Applies gradient descent to optimize parameters
  • Updates weights and biases to reduce error
4. Try Again
  • Uses updated parameters for next prediction
  • Processes new examples
  • Continues cycle until accuracy improves

Back Propagation: Learning from Mistakes

Back propagation is the process where the network traces back through its decisions to figure out which weights need adjustment. It's like retracing your steps to find where you went wrong.

Real Example: House Price Prediction

  • Error: $50,000 too low
Adjustment Process:
  1. Calculate error ($50,000)
  2. Trace back through network
  3. Adjust weights that caused error
  4. Try new prediction
